# DNA GC Content Analyzer using NumPy

This is a simple bioinformatics project built with Python that analyzes a DNA sequence and calculates nucleotide frequencies and GC content.

The project demonstrates how Python and NumPy can be used for basic biological data analysis.

## Features
- DNA sequence analysis
- Nucleotide frequency calculation (A, T, G, C)
- GC content calculation
- Beginner friendly bioinformatics project

## Technologies Used
- Python
- NumPy

## Project Structure
dna-gc-analyzer
│
├── gc_analyzer.py
└── README.md

## How It Works
The program performs the following steps:

1. Takes a DNA sequence as input
2. Converts the sequence into a NumPy array
3. Counts nucleotides (A, T, G, C)
4. Calculates GC content percentage
5. Displays the results

## Example Input

ATGCGCGTAACGTTAGCGCGATTA

## Example Output

DNA Sequence: ATGCGCGTAACGTTAGCGCGATTA  
Length: 24  

A: 6  
T: 5  
G: 7  
C: 6  

GC Content: 54.16 %
